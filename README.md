#  Portafolio de Retos: Semana 1 a 12


Este repositorio centraliza los proyectos y actividades desarrollados durante las primeras cinco semanas de formación en **Ciencia de Datos**. Aquí se encuentran documentados los avances en computación de alto rendimiento, análisis estadístico y desarrollo de software.

##  Estructura del Repositorio

El contenido está organizado por carpetas, cada una correspondiente a una semana de retos: ademas de que se secciono en dos partes semana 1-5 y semanas 6-12 esto para facilitar la busqueda y lectura del los retos 


##  Tecnologías Utilizadas

*   **Lenguajes:** Python, C (HPC).
*   **Herramientas:** Git, Visual Studio Code.
